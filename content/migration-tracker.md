## Migration Notes

- Source is READ-ONLY — never modify files in `old_hugo_working_copy/`
- For each event:
  1. Read `index.en.md` frontmatter
  2. Check `draft: false` — if true, mark SKIPPED
  3. Check `ecthrwatch.org` in `CONF_websites` — if not, mark SKIPPED
  4. Convert multiline `CONF_websites` list → space-separated string
  5. Replace Hugo shortcodes with MDX components:
     - `{{< dict X Y >}}` → `<Dict file="X" entry="Y" />`
     - `{{< mydict2 person X >}}` → `<Person entry="X" />`
     - `{{< original lang type file >}}` → `<Original lang="lang" type="type" file="file" />`
     - `{{< imgproc2 file Fit "..." >}}` → `<ImgProc src="file" />`
     - `{{< signature_shortcode_1_ >}}` → `<Signature />`
  6. Convert `<br>` → `<br/>` (self-closing)
  7. Destination folder name: `YYYY-MM-DD-<slug>` (from frontmatter slug field)
  8. Destination: `src/content/timeline/<folder>/index.mdx`
  9. Copy assets (images/PDFs) to both `src/content/timeline/<folder>/` and `public/timeline-of-events-and-open-letters/<slug>/`
  10. Commit with `[migrate] Event: <short-description>`
